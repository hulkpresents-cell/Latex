১. কেন XeLaTeX বা LuaLaTeX ব্যবহার করবেন?

বাংলা ইউনিকোড ফন্ট এবং যুক্তবর্ণ সুন্দরভাবে উপস্থাপনের জন্য ক্লাসিকাল PDFLaTeX এর বদলে XeLaTeX বা LuaLaTeX ব্যবহার করা আবশ্যক। এটি আপনার সিস্টেম ফন্টগুলো সরাসরি ল্যাটেক্স ফাইলে কম্পাইল করতে সাহায্য করে।

২. ওভারলিফ (Overleaf)-এ বাংলা লেখার উপায়:

  1. প্রথমে আপনার Overleaf প্রজেক্টের বাম পাশের মেনুবার (Menu)-এ ক্লিক করুন।
  2. Compiler অপশনটি পরিবর্তন করে XeLaTeX সিলেক্ট করুন।
  3. নিচের উদাহরণ কোডটি ব্যবহার করুন যেখানে গুগল ফন্ট থেকে Noto Sans Bengali লোড হবে:
\documentclass{article}
\usepackage{fontspec}
\usepackage[bangla]{babel}

% নিশ্চিত করুন যে আপনার সিস্টেমে ফন্টটি ইনস্টল করা আছে
\babelfont{rm}{Noto Serif Bengali}

\begin{document}
বাংলা ভাষা বিকাশের ইতিহাস অত্যন্ত সমৃদ্ধ।
\end{document}

৩. আপনার কম্পিউটারে অফলাইন কম্পাইলেশন:

অফলাইনে আপনার উইন্ডোজ বা ম্যাক কম্পিউটারে বাংলা ল্যাটেক্স কম্পাইল করতে চাইলে নিচে পদ্ধতি অনুসরণ করুন:

  • TeX Live: উইন্ডোজ ও ম্যাকের জন্য সম্পূর্ণ TeX Live ডাউনলোড করুন।
  • বাংলা ফন্ট: গুগল থেকে Noto Serif Bengali বা Hind Siliguri ডাউনলোড করে কম্পিউটারে ইনস্টল (Install) করুন।
  • এডিটর: আপনি TeXstudio, VS Code বা TeXworks ব্যবহার করতে পারেন এবং ফাইলটি অবশ্যই XeLaTeX ইঞ্জিনের মাধ্যমে বিল্ড করবেন।
